Python package "xlwt"

Purpose: Provide a library for developers to use to generate Microsoft Excel (tm) spreadsheet files.

Author: John Machin, Lingfo Pty Ltd (sjmachin@lexicon.net)

Licence: BSD-style (see licences.py)

Version of xlwt: 0.7.0a4

Version of Python required: 2.3 or later.

External modules required:

The package itself is pure Python with no dependencies on modules or packages outside the standard Python distribution.

Versions of Excel supported: 2004, 2002, XP, 2000, 97, 95

Quick start: 

import xlwt
from datetime import datetime

font0 = xlwt.Font()
font0.name = 'Times New Roman'
font0.colour_index = 2
font0.bold = True

style0 = xlwt.XFStyle()
style0.font = font0

style1 = xlwt.XFStyle()
style1.num_format_str = 'D-MMM-YY'

wb = xlwt.Workbook()
ws = wb.add_sheet('A Test Sheet')

ws.write(0, 0, 'Test', style0)
ws.write(1, 0, datetime.now(), style1)
ws.write(2, 0, 1)
ws.write(2, 1, 1)
ws.write(2, 2, xlwt.Formula("A3+B3"))

wb.save('example.xls')

Installation:

  • Any OS: Unzip the .zip file into a suitable directory, chdir to that directory, then do "python setup.py install".
  • If PYDIR is your Python installation directory: the main files are in PYDIR/Lib/site-packages/xlwt, docs are in the doc subdirectory.
  • If os.sep != "/": make the appropriate adjustments.
